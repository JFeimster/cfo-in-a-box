# Product Roadmap

## Phase 1: Public site + GPT Actions API

Build the product website, legal pages, no-auth API routes, calculators, scoring, generators, and OpenAPI schema.

## Phase 2: Interactive calculators and lead magnet capture

Add real client-side calculator interactions, lead capture, analytics, and CRM handoff.

## Phase 3: Starter Lab templates/community integration

Package prompts, templates, checklists, and monthly Cash Flow Clinic resources.

## Phase 4: Founder OS automation/reporting layer

Add dashboards, reminders, integrations where available, recurring reports, and structured workflow outputs.

## Phase 5: Advisor Desk human-in-the-loop support

Add monthly memo review, support calls, decision-support workflows, and operational QA.

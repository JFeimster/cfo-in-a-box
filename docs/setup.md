# Setup

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## GPT Actions setup

1. Deploy the app to Vercel.
2. Replace the OpenAPI server placeholder with the deployed URL.
3. Add the deployed `/privacy` URL to the Custom GPT privacy policy field.
4. Import `openapi.yaml` into the GPT Actions schema editor.

No authentication is required for v1.

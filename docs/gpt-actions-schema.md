# GPT Actions Schema

Use `openapi.yaml` as the starting schema for the CFO-in-a-Box Custom GPT Action.

Replace the placeholder server URL:

`https://YOUR-VERCEL-DOMAIN.vercel.app/api/cfo-in-a-box`

with the live Vercel deployment URL, or a custom API subdomain such as:

`https://api.distilledfunding.com/api/cfo-in-a-box`

Authentication: none for v1.

GPT Actions also require a public privacy policy URL. Use `/privacy` after deployment.
